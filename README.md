# QuizApp
Quiz App made with JAVA

The main Runnable class name is QuizApp. All user interaction with the app takes place here

## Features of App:
  * Registration and login
  * Ability to participate in all quizzes
  * Ability to create user's own quizzes with a huge number of questions and options
  * Ability to check user's own high scores for passed quizzes
  * Ability to check global high scores

For database used **MySQL** from **XAMPP**
